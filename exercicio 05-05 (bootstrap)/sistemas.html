O que é um Sistema Operacional?

Um sistema operacional (SO) é um software que faz a interface entre o hardware e o usuário. Ele gerencia os recursos do computador, como processador, memória, armazenamento e dispositivos de entrada e saída. Sem um sistema operacional, seria impossível utilizar um computador de forma prática.


Principais Funções

  • Gerenciamento de processos
  • Gerenciamento de memória
  • Gerenciamento de arquivos
  • Gerenciamento de dispositivos
  • Interface com o usuário (CLI ou GUI)
  • Segurança e controle de acesso

Tipos de Sistemas Operacionais

  • Monousuário vs. Multiusuário
  • Monotarefa vs. Multitarefa
  • Em tempo real
  • Distribuídos
  • Mobile (Android, iOS)
  • Desktop (Windows, Linux, macOS)
